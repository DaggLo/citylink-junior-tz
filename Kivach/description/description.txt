1. Необходимо сверстать макет с учетом, что страница может быть открыта
на устройствах с шириной экрана 1920px и 300px.

2. При ширине экрана 300px форма занимает всю ширину экрана, а каждый элемент
управления всю ширину формы.

3. В произвольном месте добавить на страницу кнопку "Сменить шрифт", при клике
на которую шрифт каждого элемента, содержащего текст, должен измениться на шрифт
(любой понравившийся вам) с fonts.google.com.

4. Форма при отправке должна пройти валидацию. В поле логин можно ввести только
буквы латинского алфавита, а в поле пароль только 4 цифры. Правильные учетные
данные: user 0000.

5. В случае успешной авторизации должно появиться всплывающее окно с приветствием.
Аналогичные окна должны всплывать в случае некорректности введенных данных,
с указанием ошибок. В каждом окне должен быть "крестик", при клике на который
сообщение должно исчезнуть. Для оформления "крестика" должен использоваться
иконочный шрифт Font Awesome, символ которого вставлен как псевдоэлемент.
